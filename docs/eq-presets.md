## Per-Band GAIA Encoding (Reverse-Engineered from btsnoop)

Each band is sent as a separate GAIA packet via cmd `0x0E01`, vendor `0x001D`:

```
FF 04 00 09 00 1D 0E 01 [header][mg_hi][mg_lo][freq_hi][freq_lo][gain_hi][gain_lo][q_hi][q_lo]
```

| Field | Encoding |
|-------|----------|
| header | `(bandCount << 4) \| (bandIndex + 1)` |
| masterGain | `(int)(masterGain * 60)` as 16-bit BE signed |
| freq | `(int)(freq * 3)` as 16-bit BE unsigned |
| gain | `(int)(gain * 60)` as 16-bit BE signed |
| q | `(int)(q * 4096)` as 16-bit BE unsigned |

Send one packet per band with 50ms delay between bands.
